## ⚡文件夹说明
- **左右3D电影：** 可挂载在智能投影kodi中在线观看,但是由于带宽原因建议下载在硬盘内观看。
- **阿里云盘：** 个人云盘资源,可电脑网页在线观看或者挂载在智能电视/盒子/投影中使用,支持打包下载。
- **首页目录：** 部分在首页已经列出的视频分类均来自互联网,优点自动更新,缺点可能部分少资源会缺失。
- **个人文件：** 无需理会。
- **手机使用：** 手机在网页可能无法使用,需要使用nPlayer等软件播放(挂载方法和KODI差不多)
